#  Multi-Modal & Research LLM Agent

A Streamlit app that combines **Google Gemini** (for text and summarization) and **Hugging Face Stable Diffusion XL** (for image generation).  
It also includes an automated **research assistant** that searches scholarly papers on arXiv and summarizes them using Gemini.
